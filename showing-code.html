Showing Code on a Web Page!

To show example Code Use

<pre><code> and close </code></pre> - and paste the code between tags.

This is an example using a CSS file, but any code will work the same.



*   {
    color: red;
    background-color: #000810;
}

h1  {
    text-align: center;
}

h2  {
    text-align: center;
}

a  {
    text-decoration: none;
    color: white;
}

li  {
    padding: 4px;
    font-size: 1em;
}

header {
 /*   font-size: 4em; */
    padding: 20px;
}
