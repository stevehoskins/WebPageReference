Should there be separate files for CSS, JS and Images?

The simple answer is YES!

They don't need to be but:-

This makes the code easier to read and maintain.

It also means that a lot of common code

doesn't have to be copied and modified to each page

every time a change is made.

For similar reasons it is a good practice to keep

CSS, Javascript and image files in separate folders.
