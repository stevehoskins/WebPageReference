Using Github and git!

Github is a popular code repository for storing, version control and distributing code.

  • Go to https://github.com/ and register.
  • Create a repository.
  • Download and Install Node.js and NPM!
  • Install git from the git website!
  • In Command mode - Do a git clone from Github!
  • Use the cloned folder to develop the site!
  • Periodically - in Command mode:
    • git status - do this at each stage!
    • git add .
    • git commit -m "comments"
    • git push
  • From Github pages got link to your online pages!
